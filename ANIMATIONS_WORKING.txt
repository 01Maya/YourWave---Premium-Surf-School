⚡ ANIMATION SYSTEM BREAKDOWN:

Initial State (Before Scroll):
├─ .slide-in-left { opacity: 0; transform: translateX(-100px); }
├─ .slide-in-right { opacity: 0; transform: translateX(100px); }
├─ .slide-in-up { opacity: 0; transform: translateY(100px); }
└─ .slide-in-down { opacity: 0; transform: translateY(-100px); }

On Intersection (Scroll Detected):
├─ useScrollAnimation hook detects visible element
├─ Finds all [class*="slide-in"] elements in section
├─ Applies staggered animation delays (0.1s between each)
└─ Adds .animate-on-scroll class to trigger animations

Animation Playback:
├─ .animate-on-scroll triggers actual keyframe animation
├─ Transforms element back to normal position (0, 0)
├─ Fades in opacity to 1
├─ Duration: 0.7s desktop, 0.6s mobile
└─ Timing: ease-in-out for smooth feel

Mobile Override (@media max-width: 768px):
├─ Smaller initial distance (50px instead of 100px)
├─ Faster animation (0.6s instead of 0.7s)
└─ Optimized for mobile screens
