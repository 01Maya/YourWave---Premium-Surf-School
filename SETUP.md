## Quick Start Guide

### Prerequisites
- Node.js 18+ installed
- pnpm installed (or use npm/yarn)

### Setup & Run in VS Code

```bash
# Open Terminal in VS Code (Ctrl + `)

# 1. Install dependencies (only first time)
pnpm install

# 2. Run development server
pnpm dev

# That's it! The site will be available at http://localhost:3000
```

### If using npm instead of pnpm
```bash
npm install
npm run dev
```

### If using yarn
```bash
yarn install
yarn dev
```

### Stop the server
Press `Ctrl + C` in the terminal

### Build for production
```bash
pnpm build
pnpm start
```

## Animations in Each Section

- **Hero**: Slide-in text from left, image from right, floating effect
- **Features**: Staggered slide-in cards with hover transforms
- **Tips Banner**: Slide-up container with animated tags
- **Who For**: Dual-column slide-in layout with center illustration
- **Gallery**: Staggered slide-up images with hover zoom
- **FAQ**: Slide-in accordion with smooth transitions
- **Contact**: Form fields with slide-in animations
- **Footer**: Staggered link animations

All animations work smoothly on both desktop and mobile!
