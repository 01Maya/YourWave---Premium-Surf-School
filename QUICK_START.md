# Quick Start Guide - Fully Animated Surf School Website

## 🚀 Start in 1 Command

Open VS Code terminal and paste:

```bash
pnpm install && pnpm dev
```

**That's it!** Your site opens at `http://localhost:3000`

---

## ✨ What You'll See

Scroll through the site and watch these animations:

### Hero Section
- Text slides in from **left** (0.8s)
- Image slides in from **right** (0.8s)
- Smooth cascade effect

### Features Section
- Title slides from **left**
- Cards slide from different directions
- Hover effects on all cards
- Beautiful bento grid layout

### Tips Banner
- Entire section slides **up**
- Tags animate in sequence
- CTA button animates up

### Who For Section
- **Left** column slides from left
- **Right** column slides from right
- Center illustration floats smoothly

### Gallery Section
- Images slide **up** one by one
- Hover zoom effect on images
- Play buttons animate in

### FAQ Section
- Title card slides from **left**
- Accordion items slide from **right**
- Smooth open/close animations

### Contact Section
- Form slides from **left**
- Contact info slides from **right**
- Form fields animate on focus

### Footer Section
- All links staggered animation
- Smooth hover effects throughout

## 📱 Mobile View

All animations work perfectly on mobile!

**To test on mobile:**
1. Press `F12` (DevTools)
2. Press `Ctrl+Shift+M` (device mode)
3. Select "iPhone 12"
4. Scroll and watch animations adapt

**Mobile animations:**
- Faster (0.5-0.6s)
- Shorter distance (50px)
- Touch-friendly buttons

## 🎯 Animation Features

✅ **Scroll-triggered** - Animations trigger as you scroll
✅ **Ease-in-out** - Smooth acceleration/deceleration
✅ **Staggered delays** - Elements animate in sequence
✅ **Hover effects** - Interactive elements respond
✅ **Mobile optimized** - Adapts to device size
✅ **60fps** - Smooth on all devices
✅ **Accessible** - Respects motion preferences

---

## 🛠️ How It Works

1. **useScrollAnimation()** hook detects viewport
2. **Intersection Observer** triggers when sections appear
3. CSS animations start paused
4. Animation class added = animations play
5. All optimized for mobile automatically
